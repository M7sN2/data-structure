# Data Structures Project

CS assignment - implementing different data structures in C++

## Data Structures

1. Array - dynamic array
2. Linked List - singly and doubly linked
3. Stack - LIFO structure
4. Queue - FIFO with priority queue
5. Binary Search Tree - BST with basic operations
6. Graph - adjacency list with BFS/DFS
7. Hash Table - chaining for collisions
8. Heap - min and max heap
9. Trie - prefix tree

## How to compile

Using g++:
```bash
g++ -std=c++11 main.cpp -o main_demo
```

Or compile individual files:
```bash
g++ -std=c++11 Array/ArrayUseCases.cpp -o array_demo
g++ -std=c++11 LinkedList/LinkedListUseCases.cpp -o linkedlist_demo
g++ -std=c++11 Stack/StackUseCases.cpp -o stack_demo
g++ -std=c++11 Queue/QueueUseCases.cpp -o queue_demo
g++ -std=c++11 Tree/TreeUseCases.cpp -o tree_demo
g++ -std=c++11 Graph/GraphUseCases.cpp -o graph_demo
g++ -std=c++11 HashTable/HashTableUseCases.cpp -o hashtable_demo
g++ -std=c++11 Heap/HeapUseCases.cpp -o heap_demo
g++ -std=c++11 Trie/TrieUseCases.cpp -o trie_demo
```

## Running

```bash
./main_demo
```

or on Windows:
```bash
main_demo.exe
```

## Examples

Each data structure has example use cases:
- Array: grades, shopping cart, temperatures
- Linked List: playlists, browser history
- Stack: expression eval, parentheses checker
- Queue: print queue, customer service
- BST: dictionary, file system
- Graph: social network, web pages
- Hash Table: student database, cache
- Heap: priority queue, job scheduling
- Trie: autocomplete, spell checker

## Requirements

- C++11 compiler
- Standard library only
